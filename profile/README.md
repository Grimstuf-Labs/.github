# Grimstuf Labs

**Mostly harmless. Occasionally experimental.**

Grimstuf Labs is an independent software lab building practical tools, automation systems and experimental engineering projects.

The work here is shaped around a simple principle: solve real problems first, then turn the useful parts into repeatable systems.

## Current focus

### Magrathea Project Factory

A local-first project factory for creating, structuring and governing software projects.

Magrathea is being built to reduce the chaos around early-stage tools by enforcing clear repository structures, repeatable standards, release discipline and safer project evolution.

### Marvin the WebExtractor

A controlled web extraction tool for structured crawling, repeatable site capture and project-based analysis.

Marvin is designed to collect useful web content in a more deliberate, reviewable way than a one-off scraper.

### Ligature Cleaner

Text cleanup for ligatures, odd characters and translation-document processing issues.

### PDF Scanner

PDF inspection, scanned-page detection, text-layer checks and security/permissions reporting.

## Project status

Some repositories are active projects. Some are controlled experiments. Some are internal foundations being prepared for wider release.

Nothing here should be treated as production-ready unless the individual repository says so.
